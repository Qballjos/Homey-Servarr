# Assets Required

This app requires the following image assets to be created:

## App Images
- `/assets/images/small.png` - 256x256px app icon (small)
- `/assets/images/large.png` - 512x512px app icon (large)
- `/assets/images/xlarge.png` - 1024x1024px app icon (xlarge)

## Driver Images
- `/drivers/servarr_hub/assets/images/small.png` - 256x256px device icon (small)
- `/drivers/servarr_hub/assets/images/large.png` - 512x512px device icon (large)

## Design Guidelines

### App Icon
- Should represent the Servarr suite (Radarr, Sonarr, Lidarr)
- Use a modern, clean design suitable for Homey
- Consider using a combined icon or a stylized "S" for Servarr

### Device Icon
- Should represent a control hub or monitoring interface
- Use colors that match the Homey design language
- Consider using a dashboard or control panel icon

## Creating Assets

You can use any image editing software to create these assets. Recommended tools:
- Figma
- Adobe Illustrator
- GIMP
- Canva

Make sure all images are PNG format with transparency support.
